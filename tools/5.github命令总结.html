    # github 总结

    # SourceTree github图像界面

    • $ git init 初始化仓库
    • $ git status 查看状态
    • $ add ./ 文件名 放入暂存区
    • $ git diff 查看文件修改的内容
    • $ git commit -m '改变的类容'
    • $ git log 查看日志
    • $ git reset --hard HEAD^ 回退一次
    • $ git reset --hard HEAD^^ 等同于 $ git reset --hard HEAD^2 回退2次
    • $ git checkout --文件名 修改撤销
    • $ rm 文件名 删除文件
    • $ git branch 分支名 创建分支
    • $ git branch 查看分支
    • $ checkout 分支名 切换分支
    • $ git merge 分支名 合并分支（合并分支当前的分支必须是marster分支）
    • $ git clone ssh

    克隆文件 ssh 是自己资源 的公钥 id_rsa 私钥
    id_ras.pub 公钥

    • cd ~ /.ssh 查看自己的公钥（必须进入在自己的目录下 ls 能看到id_rsa id_rsa.pub）
    • git branch -d 分支名 删除分支
