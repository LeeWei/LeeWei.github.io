# let、const、var区分

let 块级作用域 可以理解为{} 在{}类的

  • 为局部变量 不存在变量提升
  • 在同一作用域下不允许重复定义

const

  • 只读常量 不允许重新赋值
  • 一旦声明就必须初始化，不能只初始化不符值
  • 数组和对象所指的是内存地址不能改动，对应的值是可以修改
const arr=[]
arr.push(1)
arr[1]=2
consolt.log(arr) //[1,2]
