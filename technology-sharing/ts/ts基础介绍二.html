# 修饰符

  • public 修饰的属性或者方法是公有的，可以在任何地方被访问到，默认所有属性和方法都是public
  • private 修饰的属性或者方法是私有的，不能再声明他的类外部访问
  • protected修饰的属性或者方法是否保护的，它和private 类似，区别是在它在子类中也是允许访问的
class Animal 2{
